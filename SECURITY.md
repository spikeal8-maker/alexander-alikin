# Security Policy

Сообщения о найденных секретах, персональных данных или уязвимостях не следует
публиковать в открытом Issue.

Связь: Telegram `https://t.me/spikeal`.

- секреты хранятся только в GitHub Actions Secrets или внешнем менеджере;
- закрытые документы не копируются в публичный репозиторий;
- сведения о детях требуют отдельного разрешения;
- внешние формы и аналитика подключаются после проверки политики данных.
