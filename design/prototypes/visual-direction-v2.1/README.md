# Alexander Alikin — Visual Direction 2.1

Автономный HTML/CSS-прототип для переноса в Astro.

## Статус

Готов как визуальный и композиционный референс для интеграционного бота.
Не является production-кодом и не должен копироваться в `src/pages` целиком.

## Страницы

- `index.html` — главная;
- `business.html` — бизнес и автоматизация;
- `education.html` — образование;
- `projects.html` — проекты;
- `izo-asa.html` — проектный кейс;
- `about.html` — профиль;
- `journal.html` — редакционный раздел;
- `contact.html` — визуальная модель контакта и формы.

## Визуальная система

- графит: `#0a0f18`;
- тёплая бумага: `#f3f1eb`;
- кобальтовый акцент: `#245cff`;
- сигнальный зелёный используется только для микроакцентов;
- жёсткая editorial-сетка;
- минимальные скругления;
- без декоративных теней;
- крупные изображения и типографика;
- visualized и documentary изображения используются в разных ролях.

## Правило интеграции

Бот должен перенести дизайн-систему, компоненты и композицию в существующую Astro-архитектуру. Контент, маршруты, SEO, accessibility и Content Collections сохраняются из проекта. Статический прототип не заменяет их.
