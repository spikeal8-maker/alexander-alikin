# Как система работает целиком

1. **Накопление.** Источники поступают в приватный vault.
2. **Нормализация.** Скрипты строят реестры заметок, фактов, пробелов и каналов.
3. **Редакционное решение.** Материал получает аудиторию, формат, чувствительность
   и уровень доказательности.
4. **Экспорт.** Только approved-материал переносится в public через PR.
5. **Сборка.** Astro проверяет Markdown и генерирует статические HTML-страницы.
6. **Публикация.** Merge в `main` запускает GitHub Actions и GitHub Pages.
7. **Обновление.** Новые файлы автоматически попадают в ленты, поиск и RSS.
8. **Контроль.** Проверяются устаревание, ссылки, факты и регулярность.
