# ADR-0002: использовать Astro и GitHub Pages

- Статус: принято
- Дата: 2026-07-19

## Контекст

Основной объём системы — индексируемые тексты, проекты, биография и новости. Требуются высокая скорость, простая история изменений, переносимость Markdown и минимальная серверная поверхность.

## Решение

Использовать Astro со статической сборкой, TypeScript, content collections, GitHub Actions и GitHub Pages.

## Последствия

Сайт быстро загружается и легко индексируется. Формы, подписка и персонализированные функции потребуют отдельного сервиса и собственной модели обработки данных; до этого контакт осуществляется через официальные площадки.
