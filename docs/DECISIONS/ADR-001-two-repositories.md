# ADR-001: Разделить публичный сайт и приватный vault

Статус: принято, 19 июля 2026.

## Решение

Использовать два репозитория: public для сайта, private для исходных знаний.

## Причина

Контентный архив содержит закрытые заметки и неподтверждённые факты. Один
репозиторий создавал бы риск случайной публикации.

## Следствие

Нужен явный публикационный процесс и карта соответствия source → public.
