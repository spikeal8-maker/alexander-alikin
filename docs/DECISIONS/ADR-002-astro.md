# ADR-002: Использовать Astro и Markdown

Статус: принято, 19 июля 2026.

## Решение

Статическая генерация Astro, контентные коллекции и Markdown/MDX.

## Причина

Проект контентный, должен храниться в Git, быстро загружаться и не требовать
отдельного сервера для публикации.

## Следствие

Интерактивные функции добавляются точечно. Контентная схема становится
контрактом публикации.
