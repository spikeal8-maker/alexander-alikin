# ADR-003: Встроить доказательность в frontmatter

Статус: принято, 19 июля 2026.

## Решение

Каждый публичный материал получает `evidenceStatus`.

## Причина

В исходной базе есть сильные, но пока самоподтверждённые числовые утверждения.

## Следствие

`requires-verification` допускается только в draft. Пользователь сайта видит
границу между проверенным фактом и авторской позицией.
