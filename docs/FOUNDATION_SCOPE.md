# Объём первой архитектурной версии

Первая версия создаёт рабочую систему, а не окончательную публичную биографию.

## Входит

- архитектура публичного сайта;
- модель контента;
- главная, биография, проекты, истории, мысли, статьи, новости, поиск и контакты;
- правила доказательности и публичности;
- GitHub Actions для проверки и публикации;
- начальные материалы, основанные на подтверждённых пользователем данных и открытых проектах.

## Не входит до дополнительной проверки

- финансовые показатели;
- крупные числовые достижения;
- клиентские и партнёрские детали без разрешения;
- фотографии детей без подтверждённого права публикации;
- сгенерированный портрет;
- автоматический доступ публичного сайта к приватному vault.
