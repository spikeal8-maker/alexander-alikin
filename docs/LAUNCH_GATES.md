# Условия перехода к публичному запуску

## Gate 1 — идентичность

- утверждён канонический вариант имени;
- утверждена краткая формула текущей роли;
- загружен реальный портрет;
- социальные профили используют согласованные имя, описание и ссылку.

## Gate 2 — доверие

- опубликованы минимум три содержательных проекта;
- у каждого проекта ясно указана роль Александра;
- нет неподтверждённых крупных чисел;
- подготовлены внешние подтверждения либо честная маркировка авторского опыта.

## Gate 3 — активность

- раздел «Сейчас» заполнен;
- опубликованы минимум одна статья, две истории и несколько мыслей;
- создан реалистичный редакционный план;
- определён ответственный за обновление.

## Gate 4 — техника

- сборка проходит без ошибок;
- GitHub Pages настроен на GitHub Actions;
- собственный домен подтверждён и подключён;
- настроены canonical, sitemap, RSS и метаданные;
- мобильная версия, доступность и ссылки проверены.

## Gate 5 — сотрудничество

- описаны подходящие задачи;
- есть понятный основной контакт;
- объяснён первый шаг;
- есть процесс обработки входящего обращения;
- не обещается результат, который нельзя измерить или подтвердить.
