# Система измерения

## Главная метрика

Не число просмотров само по себе, а количество содержательных обращений, в которых посетитель может назвать конкретную задачу, релевантный материал и причину доверия.

## Диагностические метрики

- брендовые поисковые запросы;
- доля управляемых результатов в выдаче по имени;
- переходы из социальных профилей;
- просмотры биографии и проектных страниц;
- дочитывания статей и историй;
- переходы от материала к контакту;
- число квалифицированных обращений;
- темы, которые чаще приводят к сотрудничеству;
- доля устаревших материалов, пересмотренных в срок.

## Ограничение

Аналитика подключается только после выбора системы, допустимой по требованиям обработки данных. До этого GitHub, серверные логи платформы и ручной реестр обращений используются без скрытого профилирования посетителей.
