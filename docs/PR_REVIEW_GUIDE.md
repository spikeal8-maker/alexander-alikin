# Что проверить в foundation Pull Request

1. Сборка Astro проходит без ошибок.
2. На главной нет неподтверждённых рекламных цифр.
3. Биография не раскрывает закрытые сведения.
4. Все ссылки соответствуют официальным площадкам.
5. Google Cloud Console не представлен как YouTube.
6. Проекты ясно отделяют текущие продукты от исторических кейсов.
7. Публичный репозиторий не содержит исходного приватного архива.
8. GitHub Actions публикует только результат сборки из `main`.
9. Документация достаточна для дальнейшей работы без переписки.
10. До merge зафиксированы известные ограничения и необходимые входные материалы.
