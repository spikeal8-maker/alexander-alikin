# Быстрая схема публикации

```text
private vault
  → классификация
  → CLAIM / SOURCE / PERMISSION
  → редакционный черновик
  → решение Александра
  → локальный экспортный пакет
  → content/* branch в public repo
  → Pull Request
  → автоматические проверки
  → предпросмотр
  → merge в main
  → GitHub Actions
  → GitHub Pages / собственный домен
  → распространение ссылки
```

Ни один шаг не заменяет решение о публичности. Автоматизация ускоряет проверку и сборку, но не выдаёт разрешение на публикацию.
