# Чек-лист выпуска

## Содержание

- [ ] Первый экран отвечает: кто Александр, в чём польза, какой следующий шаг.
- [ ] Новые проекты показывают роль, период, результат и ограничения.
- [ ] Биографические утверждения согласованы с реестром фактов.
- [ ] Раздел «Сейчас» отражает реальную текущую деятельность.
- [ ] Нет просроченных анонсов и неработающих ссылок.

## Публичность

- [ ] Нет материалов из приватного vault без редакционного преобразования.
- [ ] Проверены персональные данные и права на изображения.
- [ ] Клиентские и партнёрские названия разрешены к публикации.
- [ ] В репозитории нет секретов.

## Техника

- [ ] `npm run validate:content`
- [ ] `npm run check`
- [ ] `npm run build`
- [ ] Проверены мобильная версия, клавиатурная навигация и контраст.
- [ ] Проверены canonical, sitemap, RSS и schema.org.

## После выпуска

- [ ] Открыта опубликованная страница.
- [ ] Проверены основные ссылки.
- [ ] Обновление распространено через релевантные площадки.
- [ ] Указана дата следующего пересмотра фактов.
