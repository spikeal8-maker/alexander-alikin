# Карта публичного репозитория

```text
.github/                  GitHub Actions, шаблоны задач и Pull Request
public/                   файлы, которые копируются в корень опубликованного сайта
src/components/           переиспользуемые интерфейсные блоки
src/content/              утверждённые проекты, истории, мысли, статьи и новости
src/data/                 навигация, контакты, хронология и конфигурация
src/layouts/              общие каркасы страниц
src/pages/                URL-маршруты сайта
src/styles/               дизайн-система и глобальные стили
docs/                     стратегия, архитектура, правила и решения
scripts/                  проверки контента и служебные генераторы
tests/                    автоматические проверки поведения
```

## Правило размещения

- контент не кодируется прямо в компонентах, если он должен регулярно обновляться;
- изображения не дублируются в нескольких каталогах;
- крупные документы публикуются только при реальной ценности для посетителя;
- конфиденциальные источники не упоминаются путями или именами файлов;
- документация объясняет не только что настроено, но и почему принято решение.
