# Реестр основных рисков

| Риск | Последствие | Контроль |
|---|---|---|
| Публикация неподтверждённой цифры | потеря доверия и репутационный спор | proofLevel, CLAIM, ручное решение |
| Утечка приватной заметки | вред владельцу и третьим лицам | два репозитория, экспортный разрыв, PR |
| Размытое позиционирование | посетитель не понимает пользу | одна главная формула и маршруты аудиторий |
| Неактуальный сайт | выдача показывает устаревшую роль | раздел «Сейчас», даты ревизии, backlog |
| Смешение личного бренда и тематических экспериментов | снижение профессиональной ясности | отдельные проектные контуры |
| Зависимость от одного интерфейса или подрядчика | сложное развитие и миграция | открытые форматы, Markdown, Git |
| Отсутствие реальных фотографий | обезличенность и недоверие | обязательный фото-пакет до запуска |
| Автоматическая публикация низкокачественного контента | информационный шум | редакционный ритм вместо контент-фабрики |
| Неработающие внешние ссылки | снижение доверия и SEO | автоматическая и квартальная проверка |
| Сбор персональных данных без модели обработки | правовой и технический риск | без серверных форм на первом этапе |
