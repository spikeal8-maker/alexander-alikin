# Дорожная карта

## 0. Фундамент
Два репозитория, Astro, документация, CI/CD, схемы, поиск, RSS, sitemap, JSON-LD.

## 1. Достоверная личность
Портрет, каноническая биография, временная линия, единые профили, медиакит.

## 2. Сильные проекты
ИЗО АСА, инженерное образование, робототехника, ИИ/автоматизация, Лига роботов,
Laser Love — каждый после проверки.

## 3. Авторское медиа
Пять историй, двенадцать мыслей, три статьи, видео, «Сейчас», редакционный цикл.

## 4. Доверие и сотрудничество
Достижения, отзывы, внешние публикации, бриф, страницы для школ и партнёров.

## 5. Второй мозг
Публичная карта тем, связанные заметки, семантическая навигация и ограниченный
ИИ-помощник только по опубликованной базе.

## 6. Измерение
Search Console, Яндекс Вебмастер, аналитика, аудит выдачи и устаревших страниц.
