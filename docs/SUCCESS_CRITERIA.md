# Критерии успеха foundation-версии

## Понимание

Пять независимых посетителей после десяти секунд просмотра могут назвать:

- имя;
- текущую профессиональную роль;
- одну понятную пользу;
- основное действие.

## Доверие

Посетитель может найти минимум три проекта с ясной ролью Александра, увидеть природу доказательств и отличить авторское суждение от документально подтверждённого факта.

## Интерес

На сайте есть достаточно материалов, чтобы пройти минимум три смысловых перехода: история → проект → мысль или статья → биография → сотрудничество.

## Актуальность

Раздел «Сейчас», контакты, активные проекты и главная формула пересмотрены в установленный срок.

## Управляемость

Новый материал можно выпустить через GitHub без изменения архитектуры: Markdown → PR → проверки → merge → публикация.

## Безопасность

Публичная сборка не зависит от приватного репозитория и не содержит секретов, закрытых путей, внутренних комментариев и неразрешённых материалов.
