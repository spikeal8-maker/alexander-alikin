# Карта внедрения изображений

## Принцип

Папка `visualized` содержит AI-визуализации на основе фотографий Александра.
Они подходят для портретов, hero и иллюстративных блоков, но не являются
документальным доказательством конкретного события.

Папка `documentary` содержит обработанные реальные фотографии.
Их использовать для биографии, истории проектов, Лиги роботов, Laser Love
и других подтверждаемых эпизодов.
