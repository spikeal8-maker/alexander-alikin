## 3. Численные утверждения (заблокированы)

Нижеперечисленные утверждения имеют статус `needs_evidence` и `publishable: false`. Для каждого указано, какое доказательство разблокирует публикацию.

| Claim ID | Утверждение | Что требуется для разблокировки |
|---|---|---|
| CL-005 | «Пилотный запуск в 2025 году. Платформа охватывает 5 дисциплин» | PROJ-002 с подтверждённым списком дисциплин и датой пилотного запуска |
| CL-007 | «Более 10 лет практики» | BIO-001 с подтверждением даты начала педагогической деятельности |
| CL-008 | «С 2017 года веду практические занятия» | BIO-001 или DOC-001 (фото 2017 года) с подтверждением года начала занятий |
| CL-009 | «Сотни учеников» | PROJ-001 с документальным подтверждением (списки групп, журналы, отчёты). Альтернатива: заменить на «ряд учеников» |
| CL-010 | «Десятки поступили в ведущие технические вузы» | PROJ-001 с подтверждением: обезличенный список, названия вузов, годы. Высокий репутационный риск. Альтернатива: удалить |
| CL-014 | «Подготовлены десятки преподавателей» | PROJ-001 с подтверждением количества педагогов, форматов подготовки, периода |
| CL-023 | «Ученики успешно выступают на соревнованиях» | PROJ-001 с перечнем соревнований, дат, достижений или BIO-002 с публичными ссылками на результаты |
| CL-027 | «Сроки пилота: 4–8 недель» | Минимум один завершённый пилотный проект с подтверждёнными сроками |
| CL-032 | «Аудит: 2–4 недели» | Реальные данные о длительности проектов по аудиту |

**Решение владельца:** Для каждого утверждения выбрать один из вариантов:
1. Предоставить запрошенное доказательство → повысить до `publishable: true`
2. Заменить количественное утверждение на качественное (например, «сотни» → «ряд», «десятки» → «несколько») → пересмотреть формулировку
3. Удалить утверждение полностью → исключить из page-texts и claims
